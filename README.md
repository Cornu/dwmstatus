dwmstatus
=========

![](./dwmstatus.png)

> Conky like DWM status bar written in C.

- CPU Load / Freq
- Temperature
- Memory
- Wireless Network
- Wired Network
- Battery
- Volume
- Clock

---

### /tmp files

- "/tmp/volume" volume in %
- "/tmp/wireless" if exists contains essid
- "/tmp/wired" if exists network is up
